# Week 1 - Python Basics

Current reality: I am starting from almost zero Python.

So Week 1 is not about building the full monitor immediately. It is about becoming able to read, change, run, and explain small Python scripts.

## Week 1 Goal

By the end of this week, I should be able to:

- Run Python scripts from PowerShell.
- Use variables, strings, numbers, lists, and dictionaries.
- Use `if/else`.
- Use `for` loops.
- Write and call small functions.
- Read one command line argument.
- Make at least one Git commit.

## Day 0

File:

- `scripts/python_basics_day0.py`

Tasks:

- Run the file once.
- Change `name`.
- Change `age`.
- Add one item to the `tasks` list.
- Run it again.
- Explain what `if is_adult(age):` means.

Command:

```powershell
python .\scripts\python_basics_day0.py
```
